# UI/UX Pro Max Design Intelligence System

This skill provides industry-grade design guidelines, visual aesthetics, micro-interactions, color palettes, and component patterns to turn any application into a stunning, premium product.

## 🎨 Core Color Palette & Visual System

1. **Dark Mode Excellence**:
   - Primary Background: `#0f172a` (Deep Slate / Navy Dark)
   - Surface Layer 1: `#18181b` (Charcoal Card)
   - Surface Layer 2: `#27272a` (Elevated Element)
   - Primary Brand Accent: `#ccff00` (Electric Neon Lime - high contrast, high energy)
   - Secondary Accent: `#3b82f6` (Royal Blue / Electric Blue for royal badges)
   - Highlight Cyan: `#38bdf8` (Light Sky Blue for user GPS markers & live pulses)
   - Alert Amber: `#f59e0b` (Warning / Returning status)
   - Danger Red: `#ef4444` (Cancellation / Expiry)

2. **Glassmorphism & Depth**:
   - Backdrop Blur: `backdrop-filter: blur(16px)`
   - Border Tints: `1px solid rgba(204, 255, 0, 0.2)`
   - Multi-layer Drop Shadows: `box-shadow: 0 10px 30px rgba(0,0,0,0.6)`

3. **Typography & Hierarchy**:
   - Modern System Font Stack: `Inter`, `-apple-system`, `BlinkMacSystemFont`, `Roboto`, `sans-serif`
   - Weight Contrast: Heavy `900` for titles and primary action buttons; `700` for subtitles; `500` for body
   - Letter Spacing: Uppercase badges with `tracking-wider` (`letter-spacing: 0.05em`)

4. **Micro-Animations & Interactive Feedback**:
   - Hover scale: `transform: scale(1.03); transition: transform 0.2s ease;`
   - Active press scale: `transform: scale(0.97);`
   - Pulsing Ring Effects: Radial keyframe pulse for user GPS locator & live statuses

## 📱 Mobile UI Component Guidelines

- **Compact Control Bars**: Single-row horizontal scrollable filter chips with active accent background (`#ccff00`) and dark text (`#000000`).
- **Interactive Map Popups**: Dark card container with neon borders, 60px vehicle thumbnails, price per day in bold neon, and high-contrast dual buttons (`ĐẶT XE NGAY` + `💬 CHAT`).
- **Status Badges**: Curved pill badges (`border-radius: 9999px`) with 15% opacity background fill and matching solid text/border color.
